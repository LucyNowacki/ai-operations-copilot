# Portfolio and GitHub Analysis

This project is grounded in Lucy Nowacki's public portfolio and GitHub:

- Portfolio: <https://lucynowacki.github.io>
- GitHub: <https://github.com/LucyNowacki>

## Main Themes

Lucy’s public work combines applied AI, mathematical modelling and practical product thinking:

- **Medical AI and wound diagnostics:** portfolio material on AI-powered wound detection, mobile diagnosis and deployable healthcare tools; GitHub projects include `CancerDetector` and `AzureCancerDetector`.
- **Recommender systems and semantic search:** `Semantic-Recommender` uses sentence embeddings, clustering, dimensionality reduction and evaluation metrics for arXiv paper discovery.
- **Financial modelling and risk:** portfolio posts cover option pricing, explicit and implicit Method of Lines, crypto analytics with Pandas/Plotly and mathematical finance.
- **Climate modelling and DMD:** portfolio work on Dynamic Mode Decomposition for identifying coherent climate structures.
- **PINNs, numerical PDEs and scientific ML:** portfolio material covers Physics-Informed Neural Networks, heat-equation examples and numerical validation.
- **NLP, hate-speech detection and Azure deployment:** public repos include Azure hate-detector deployment and xLSTM hate-speech experiments.
- **Robotics, drones, prosthetics and AI-agent workflows:** portfolio messaging covers UAV navigation, modular prosthetics, defence/robotics contexts and agent-first operational solutions.
- **Robustness and evaluation:** adversarial attacks, SVD/PCA, robust PCA and ML-from-scratch work show interest in model behaviour and failure modes.

## Strongest Employability Signals

- Mathematical AI judgement: useful for model evaluation, uncertainty, numerical methods and technical credibility.
- Applied product framing: medical, fintech, recommender, robotics and climate examples make the work understandable to employers.
- Python/data workflow evidence: GitHub projects show notebooks, deployable apps, CSV ingestion and analytics.
- Technical communication: portfolio posts explain advanced concepts to readers, which is valuable for AI operations, DevRel, analytics and startup roles.

## How The Themes Shape This Product

The Career Proof Engine uses Lucy’s themes as demo business domains rather than generic examples. Each example note becomes a project brief with objective, requirements, business value, risks, dependencies, owner, due date, stakeholder questions and next actions.

The result demonstrates a modern employability story: Lucy can connect AI assistants, structured extraction, validation, review gates, SQL storage, dashboards and n8n-style automation into an auditable workflow for real operational decisions.
