# Climate-Risk DMD Modelling Brief

Source: Lucy portfolio post on Dynamic Mode Decomposition for El Nino discovery and climate modelling.

Owner: Lucy Nowacki
Target Date: 2026-08-01

Research notes: build a launch brief for a climate-risk modelling workflow using Dynamic Mode Decomposition. The workflow should ingest time-series or gridded climate data, identify coherent modes, summarise uncertainty and expose stakeholder questions for a risk or research team. The dashboard should show status, open risks and model interpretation notes.

The system must make source checking explicit because DMD patterns can look convincing even when the input data or assumptions are weak.
